# Keeper App

A note-taking web application built with React, inspired by Google Keep.

## Live Demo

🔗 [keeper-app-zeta-five.vercel.app](https://keeper-app-zeta-five.vercel.app)

## GitHub Repository

🔗 [github.com/ve-ltn/keeper-app](https://github.com/ve-ltn/keeper-app)

## Features

- Add new notes with a title and content
- Delete notes when no longer needed
- Clean and responsive UI

## Tech Stack

- React
- JavaScript
- CSS
- Vercel (deployment)

## Try It Out

Visit the live demo above to start creating and managing your notes directly in the browser.
